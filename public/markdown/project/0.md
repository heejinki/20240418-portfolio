- `메인 페이지`와 `이력서 CRUD`를 담당하였으며 신입 개발자의 이력서 작성을 용이하게 하기 위해 `직관적인 UI 제작`, `인풋 요소들을 효율적으로 배치`하였습니다.

- 디자이너 팀원의 부재로 메인 페이지 및 이력서 `디자인`을 직접 제작하였습니다.
